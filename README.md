# HR-ATTRITION-ANALYSIS-DASHBOARD
This project analyzes employee attrition and workforce trends using Power BI. The dashboard helps HR teams understand factors influencing employee turnover and provides insights into employee demographics, performance, satisfaction, and compensation.
# Objectives
- Monitor employee attrition trends
- Identify key factors contributing to attrition
- Analyze employee demographics and performance
- Support data-driven HR decision-making
# Tools Used
- Power BI
- Power Query
- DAX
- SQL Server
# SQL Data Cleaning
- Removed duplicate records
- Trimmed leading and trailing spaces using TRIM()
- Standardized text using UPPER() and LOWER()
- Replaced null values
- Corrected inconsistent values
# Power Query Transformations
- Changed data types
- Removed errors
- Handled missing values
- Renamed columns
- Created custom columns
- Split and merged columns where required
- Applied data profiling checks
- Validated data types

# Data Model

- The project follows a star schema design with:

- Fact Table: Employee, Performance Rating
- Dimension Tables: Education Level, Rating Level, Satisfaction Level, Date Table
- Key KPIs
- Total Employees
- Attrition Count
- Attrition Rate %
- Average Salary
- Average Age
- Average Manager Rating
- Average Job Satisfaction
# Features
- Interactive Filters and Slicers
- Attrition Analysis by Department
- Attrition Analysis by Job Role
- Satisfaction and Performance Insights
- What-If Parameter Analysis
- Dynamic DAX Measures
- Key Influencers Visual
- Bookmarks & Selection
